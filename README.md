# Mental Health in Medical Students — Statistical Analysis

This project presents an exploratory and statistical analysis of mental health indicators among medical students.

## Objectives

- Explore the structure of the dataset
- Perform descriptive and bivariate analyses
- Study relationships between psychological indicators
- Apply statistical methods such as ANOVA, PCA and linear regression

## Methods

The analysis includes:

- Data cleaning
- Exploratory Data Analysis
- Correlation analysis
- ANOVA
- Principal Component Analysis
- Multiple linear regression
- Model diagnostics

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Statsmodels
- Scikit-learn
- Jupyter Notebook
